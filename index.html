Breezee Weather App

This weather application dynamically displays current weather conditions as well as 15-hour forecasts in increments of 3 hours. The data are imported through Open Weather Map’s One Call API, which uses geographical coordinates to collect the weather information. When the user types in a city name in the search box, this app converts the name into latitude and longitude using JavaScript, and then sends that info back to Open Weather Map.

I styled the app with CSS3 and adopted Neumorphism for the overall look.
